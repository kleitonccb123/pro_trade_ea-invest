Browser:
  1. Abrir http://localhost:5173
  2. Login (criar conta se necessário)
  3. Navegar para http://localhost:5173/kucoin
  4. Preencher formulário:
     - API Key: qualquer texto com 10+ chars
     - API Secret: qualquer texto com 20+ chars
     - API Passphrase: qualquer texto com 6+ chars
     - Sandbox: ✓ (ativado para segurança)
  5. Clicar "Conectar KuCoin"
  6. Esperar notificação verde: "✅ KuCoin Conectada com Sucesso!"

Pronto! Credenciais estão encriptadas no MongoDB.
