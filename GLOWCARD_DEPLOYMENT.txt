💡 HOW IT WORKS
═══════════════════════════════════════════════════════════════════════════

1. POINTER TRACKING:
   • Document listens to 'pointermove' events
   • Updates CSS variables: --x, --y, --xp, --yp

2. GRADIENT CALCULATION:
   • Radial gradient positioned at (--x, --y)
   • Hue calculated from --xp and spread value
   • Creates dynamic color based on position

3. DUAL GLOW:
   • ::before: Main colored glow (bright)
   • ::after: White glow (subtle highlight)

4. MASKING:
   • Mask clip: padding-box (background)
   • Mask clip: border-box (border)
   • Creates distinct border effect

  <GlowCard glowColor="blue" size="md">

    glowColor="purple"
    customSize={true}
    width={500}
    height={600}
    className="my-custom-class"
  >
    <h3>Custom Title</h3>
    <p>Custom content</p>
  </GlowCard>

WITH DYNAMIC COLOR:
  <GlowCard
    glowColor={robot.status === 'active' ? 'green' : 'orange'}
    size="lg"
  >
    {/* Content */}
  </GlowCard>


🐛 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════

Problem: Glow not visible
Solution: Check DevTools > Inspector for [data-glow] attribute

Problem: Cards wrong size
Solution: Verify size prop ('sm'|'md'|'lg') or customSize settings

Problem: Performance issues
Solution: Reduce number of visible cards or disable animations

Problem: Colors not showing
Solution: Hard refresh (Ctrl+Shift+R) and clear browser cache
