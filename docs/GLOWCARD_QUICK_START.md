# 🎯 QUICK START - GlowCard Visualization

## ⚡ 30 segundos para ver em ação

### 1. Abra o navegador
```
http://localhost:8080/robots
```

### 2. Localize a seção
```
Procure por: ⭐ ROBÔS EM DESTAQUE
Local: Entre a barra de busca e a grade de robôs
```

### 3. Veja o efeito
```
Mova o mouse sobre os cards
Veja o brilho acompanhar o cursor ✨
```
