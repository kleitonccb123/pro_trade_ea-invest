Running the Tests
=================

1. Install dependencies:
   ```
   pip install pytest pytest-asyncio pytest-mock
   ```

2. Run all tests:
   ```
   pytest backend/
   ```

3. Run with verbose output:
   ```
   pytest backend/ -v
   ```

4. Run with coverage:
   ```
   pytest backend/ --cov=app/db --cov-report=html
   ```

5. Run specific test class:
   ```
   pytest backend/test_bot_service.py::TestBotServiceCreate -v
   ```

6. Run async tests only:
   ```
   pytest backend/ -m asyncio -v
   ```

7. Run with colored output and detailed info:
   ```
   pytest backend/ -vv --color=yes
   ```

8. Run tests matching pattern:

   ```
   pytest backend/ -k "test_validate" -v
   ```


Key Testing Patterns
====================

Pattern 1: Mock Collection Setup
-------------------------------
```python
@pytest.mark.asyncio
async def test_something(self, mock_bots_collection):
    mock_bots_collection.insert_one = AsyncMock(
        return_value=MagicMock(inserted_id=ObjectId())
    )
    service = BotService(mock_bots_collection)
    # ... test code ...
```

Pattern 2: Using Fixtures with Custom Data
-------------------------------------------
```python
@pytest.mark.asyncio
async def test_something(self, sample_bot_data):
    sample_bot_data["name"] = "Custom Name"
    bot = Bot(**sample_bot_data)
    assert bot.name == "Custom Name"
```

Pattern 3: Testing Error Paths
------------------------------
```python
@pytest.mark.asyncio
async def test_update_config_blocked(self, mock_bots_collection):
    mock_bots_collection.find_one = AsyncMock(
        return_value={"status": BotStatus.RUNNING.value}
    )
    service = BotService(mock_bots_collection)
    success = await service.update_bot_config(...)
    assert success is False
```

Pattern 4: Testing Integration Points
-------------------------------------
```python
@pytest.mark.asyncio
async def test_validate_and_execute(self, mock_orders_collection):
    with patch('app.db.order_service.get_risk_manager') as mock_risk_mgr:
        mock_risk_mgr.validate_order = AsyncMock(return_value=(True, None))
        # ... test code ...
```
