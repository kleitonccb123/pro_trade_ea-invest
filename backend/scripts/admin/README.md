# Admin Scripts

Scripts administrativos e utilitários de desenvolvimento.

> ⚠️ **ATENÇÃO:** Estes scripts são para uso administrativo local.  
> Nunca execute em produção sem revisar cuidadosamente.

## Scripts Disponíveis

| Script | Descrição |
|--------|-----------|
| `_list_users.py` | Lista todos os usuários do sistema |
| `_set_plan.py` | Altera o plano de um usuário |
| `check_admin.py` | Verifica status de administrador |
| `fix_admin.py` | Corrige permissões de administrador |
| `reset_admin.py` | Reseta conta de administrador |
| `run_server.py` | Inicia o servidor de desenvolvimento |
| `start_robot_updater.py` | Inicia o processo de atualização de robôs |

## Uso

```bash
# A partir da raiz do projeto
cd backend/scripts/admin
python _list_users.py
```
