## ✨ Recursos Principais

### Design
- 🎨 Paleta profissional slate + azul
- 📐 Layout consistente
- 🔤 Tipografia hierarquizada
- 📱 Responsivo 100%
- ⚡ Performance otimizada

### Features
- 🔐 Login com email/senha
- 🌐 Login com Google
- 🤖 Gerenciamento de robôs
- 📊 Dashboard com robôs
- 🔍 Busca e filtros
- 👤 Menu de usuário
- 📱 Responsível (mobile/tablet/desktop)

### Tecnologia
- React 18 + TypeScript
- Vite + Tailwind CSS
- FastAPI + MongoDB
- JWT + bcrypt
- Shadcn/UI + Lucide
