✅ PASSO 1️⃣  : Health Check
   └─ GET /health → 200 OK

✅ PASSO 2️⃣  : Autenticação
   └─ POST /api/auth/register → 200/201/400
   └─ POST /api/auth/login → 200 + JWT Token

✅ PASSO 3️⃣  : Criação de Bot
   └─ POST /bots → 201 CREATED
   └─ Response contém: id, name, symbol

   └─ GET /bots/{id} → 200 OK
   └─ Bot persistido no banco

   └─ POST /bots/{id}/start → 200 OK
   └─ Modo simulação (sem Binance)

✅ PASSO 6️⃣  : Verificar Estado
   └─ GET /bots/{id} → is_running = True
   └─ Confirma execução

✅ PASSO 7️⃣  : Cleanup
   └─ POST /bots/{id}/stop → 200 OK
   └─ Remove dados de teste

✅ INTEGRADO : Ponta-a-Ponta
   └─ Executa todo fluxo em sequência
   └─ Identifica exatamente onde falha


🚀 COMO USAR

cd backend
python -m uvicorn app.main:app --host 0.0.0.0 --port 8000

## Opção A: Teste Integrado Completo (RECOMENDADO)
pytest tests/integration/test_happy_path.py::test_happy_path_complete -v -s

## Opção B: Todos os Testes
pytest tests/integration/test_happy_path.py -v -s

## Opção C: Versão Simplificada (sem pytest)
python tests/integration/test_happy_path_simple.py

## Opção D: Com Porta Customizada
pytest tests/integration/test_happy_path.py -v --base-url http://localhost:8001
